# 🏝️ Treasure Island Adventure

A beginner-friendly text-based adventure game built with Python.

The player explores Treasure Island by making a series of decisions. Each choice leads to a different outcome, with only one path leading to the hidden treasure. This project was created to practice Python fundamentals while learning how to structure programs using functions and input validation.

---

## 📌 Features

- Interactive text-based adventure
- Multiple story paths and endings
- Input validation to prevent invalid choices
- Replay option to play multiple times
- Modular code using functions
- Simple storytelling experience in the terminal

---

## 🛠️ Technologies

- Python 3
- Standard Python libraries (`time`)

---

## 📚 Concepts Practiced

- Variables
- Conditional Statements (`if`, `elif`, `else`)
- Functions
- Loops
- User Input
- Input Validation
- String Formatting
- Code Organization


2. Navigate to the project folder.

```bash
cd python-practice/Day03_TreasureIsland
```

3. Run the program.

