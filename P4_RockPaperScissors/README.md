# ✊ Rock Paper Scissors

A command-line implementation of the classic Rock Paper Scissors game written in Python.

## Features

- Play against the computer
- ASCII art for each move
- Input validation
- Score tracking
- Multiple rounds
- Final winner announcement

## Concepts Practiced

- Functions
- Loops
- Conditional statements
- Random module
- Exception handling
- User input
- Variables

## How to Run

```bash
python main.py
```
